🌟 Aurevia Login Page

A modern, responsive, and user-friendly authentication interface built using HTML, CSS, and JavaScript. Aurevia provides a professional login experience with real-time validation, password recovery flow, theme switching, and enhanced usability features.

🚀 Features :

🔐 Authentication Interface

Email Address input field
Password input field
Login button
Clean and modern UI design

✅ Form Validation

Real-time email validation
Error messages for invalid inputs
User-friendly feedback system

👁️ Password Visibility Toggle

Show/Hide password functionality
Improves user convenience while maintaining security

💾 Remember Me

Allows users to save login preferences
Enhances user experience during future visits

🔑 Forgot Password Flow

Dedicated password reset page
Email input for password recovery
Success confirmation screen after reset request

🎨 Theme Toggle

Light Mode and Dark Mode support
Instant theme switching
Consistent and responsive design across themes

📱 Responsive Design

Optimized for desktop, tablet, and mobile devices
Smooth and adaptive user interface

🛠️ Technologies Used

HTML5
CSS3
JavaScript (Vanilla JS)
